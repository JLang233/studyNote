# 数据结构
## 顺序表
顺序表的插入：
``` JavaScript
insert(item,pos){
    if(pos < 0 || pos > this.array.length){
        return false;
    }

    for(let i = this.array.length; i >= pos; i--){
        this.array[i] = this.array[i-1];
    }

    this.array.[pos] = item;
    return item;
}
```

顺序表的删除：
``` JavaScript
delete(pos){
    if(pos < 0 || pos > this.array.length){
        return null;
    }
    let r = this.array[pos];
    for(let i = pos; i < this.array.length-1;i++){
        this.array[i]=this.array[i+1];
    }

    this.array.lenth--;
    return r;
}
```
链表的查找：
``` JavaScript
findPos(pos){
    let r = this.head;
    for(let i = 0 ; i < pos;i++){
        r = r.next;
    }
    return r;
}
```
